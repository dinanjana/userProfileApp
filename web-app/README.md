# Userprofiles web app

## Getting Started

First, run the development server:

```bash
npm install
```

Set en variables described in .env.example

```bash
npm run dev
# or
yarn dev
```

Production

```bash
npm run build
npm start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to access to user profile web app.
